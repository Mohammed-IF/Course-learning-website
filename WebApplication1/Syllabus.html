Text book: Computer Networking: A Top-down Approach

Course Description (catalog):

This course covers: definition of computer networks and their objectives and

applications. Computer network architecture: layering, protocols and standard models,

the ISO OSI and TCP/IP reference models. It covers the application layer and its

protocols (HTTP, FTP etc). The transport protocols namely UDP and TCP. The network

layer and IP, intro to routing algorithms. Data link layer: data link layer functions and

standards, error check and correction.

Prerequisites:IS201

Co-requisite:CSC227

Course type:Core

Course Learning Outcomes (CLOs):

  1. Become familiar with layered communication architectures (OSI and TCP/IP)
  2. Learn about types of delay in packet switching
  3. Understand the client/server and P2P models and key application layer protocols
  4. Understand the need for a web cache
  5. Understand the concepts of reliable data transfer and how TCP implements them
  6. Know the principles of congestion control and trade-offs in fairness and efficiency
  7. Learn the principles of routing and the semantics and syntax of IP
  8. Understand the basics of error detection including parity, checksums, and CRC

Major Topics of the course:

  • Introduction to Computer Networks
  • Application Layer
  • Transport Layer
  • Network Layer
  • Link Layer
